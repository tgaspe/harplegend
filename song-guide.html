How to Create a Song Chart

HarpHero · ChewPacaStudios

You can upload your own songs to HarpHero using the My Songs section. Only the MIDI chart file is required — audio and cover art are optional.

File Format Required
Chart file .mid ✅ Yes
Audio file .ogg Optional
Cover art image Optional
Step 1

Find a MIDI File

Search for your song on one of these sites — there are many others, but these are the ones I use most:

  • MidiWorld — large catalogue of free MIDI files
  • BitMidi — easy search and preview
  • Enchor — pre-charted tracks, usually the best quality
Tip: Enchor songs come with a .mid file already charted to the melody. Search the song name, download the package, and look for the .mid file inside.
Step 2

Inspect the MIDI

Before editing, open the file in a MIDI viewer to understand its structure.

Signal MIDI — free, browser-based (many other MIDI editors exist)

Look for:

  • How many tracks/channels the file has
  • Which track contains the melody — the main tune, not drums, bass, or chords
  • Whether the notes look reasonable — not too dense, not empty
Step 3

Edit the MIDI

This is where you shape the chart to your liking. Common adjustments:

  • Transpose notes by X semitones to match your harmonica's key
  • Adjust note position, length, and velocity (volume)
  • Change the BPM to make the chart easier or harder
  • Change track instruments if needed
Harmonica limitations: A diatonic harmonica can only play the notes of a specific major scale. Many MIDI melodies contain notes your harmonica cannot produce. Go through the melody and remove or substitute any out-of-range notes — transpose them an octave, replace them with the nearest playable note, or remove them if they are passing tones.

To see all note data as a spreadsheet:

MIDI to CSV — convert your MIDI to a spreadsheet to inspect and fine-tune note data
Step 4

Extract the Harmonica Track

Once you are happy with the edits, save only the melody track as a separate MIDI file. This is the file HarpHero uses to drive the note lanes.

  • Recommended name: chart.mid
  • It should contain only the harmonica notes — no drums, chords, or other instruments
Step 5

Create the Audio File (optional)

If you want a backing track playing during gameplay, convert your full MIDI (all instruments, not just the melody) to OGG.

FreeConvert MIDI → OGG
  1. Upload your full .mid file
  2. Download the .ogg result
Why OGG? It is a highly storage-efficient audio format, important for mobile.
Step 6

Add Silence at the Start (optional)

If you want a lead-in before the first note hits, shift both files forward by the same amount. I use 3 seconds.

For chart.mid: Shift all MIDI notes forward by 3 seconds using a MIDI editor.

For the .ogg (only if you created one in Step 5): add matching silence to the beginning.

Add Silence — Elysia Tools
  1. Upload your .ogg
  2. Add 3000 ms of silence to the beginning
  3. Download the result
The silence must match exactly in both files — otherwise notes and music will be out of sync.
Step 7

Create the Album Art (optional)

Find a fitting image on a free stock photo site:

  • Pixabay — free, no attribution required
  • Pexels — free, no attribution required

Then design your cover in Canva or any image editor:

  • Use a 1000 × 1000 px square canvas
  • Add the song or album name in a stylish font
  • Export as JPG or PNG
Step 8

Upload to HarpHero

  1. Open HarpHero → tap My Songs → tap +
  2. Pick your chart.mid file
  3. Pick your .ogg audio file (optional)
  4. Pick your cover image (optional)
  5. Enter the song title, artist, key, and difficulty
  6. Tap Save

Your song will appear in the My Songs carousel and is ready to play.

Quick Checklist

  • chart.mid — melody track only, harmonica-playable notes
  • .ogg audio — full mix with matching lead-in silence (optional)
  • Lead-in silence matches between chart and audio (optional)
  • Cover art — 1000 × 1000 px square (optional)
